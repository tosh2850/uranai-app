2. 基本の仕組み

陰陽と五行（木・火・土・金・水）

算命学（さんめいがく）の基盤には、中国古代思想である「陰陽五行」があります。陰陽はすべての事象を「陽」と「陰」の二つの側面で捉える考え方であり、算命学の命式（めいしき）や運勢鑑定の根幹を成しています。陽は活動的・外向的な力を表し、陰は静的・内向的な力を表します。昼と夜、男性と女性、動と静など、自然界のあらゆるものは陰と陽の対立と調和で成り立っているとされます。

ここで重要なのは「陰」と「陽」の違いです。

  • 陽： 外へ広がり、直線的で積極的な性質。太陽や昼のように表に出やすく、活動的でエネルギッシュな側面を持ちます。
  • 陰： 内に向かい、曲線的で受容的な性質。月や夜のように内側に働きかけ、控えめで蓄える力を持ちます。

同じ五行でも「陽」と「陰」で表現が変わります。例えば木の「陽」である甲は大樹のようにまっすぐ伸びる力を表し、木の「陰」である乙は草花のようにしなやかで柔らかく伸びる力を示します。このように、陰陽の違いはエネルギーの方向性や表れ方の差を意味します。

五行とは「木・火・土・金・水」の五つの要素です。これは単なる自然の物質ではなく、性質や作用を示すものです。算命学では五行バランスの偏りをチェックし、性格診断や相性占い、運勢鑑定に活用します。

  • 木： 成長・発展・柔軟性。「始まり」や「計画」を意味。
  • 火： 情熱・表現・直感。「発展」「上昇」を象徴。
  • 土： 安定・調和・受容。大地のように他を支える。
  • 金： 規律・判断・切り開く力。「決断」「収穫」を意味。
  • 水： 知恵・冷静・柔らかさ。「柔軟」「浸透」「知性」を象徴。

五行は互いに関係し合い、バランスを取りながら循環しています。

  • 相生（そうしょう）： 木は火を生み、火は土を生み、土は金を生み、金は水を生み、水は木を育てる。
  • 相剋（そうこく）： 木は土を制し、土は水を塞き、水は火を消し、火は金を溶かし、金は木を切る。

算命学では、この五行の偏りや調和を見ることで「どんな性質が強いか」「どの場面で力を発揮しやすいか」を読み解き、人生設計や相性判断に活かします。

十干十二支の概要

五行と陰陽の思想を組み合わせたものが十干（じっかん）と十二支（じゅうにし）です。これらは暦や命式の基礎を形づくり、干支占いや四柱推命とも共通する基盤です。

十干（10種類）

十干は五行に「陰」と「陽」を配してできた10種類のエネルギーの型です。算命学鑑定ではこの十干を基にして「日干（日主）」を決定し、その人の核を読み取ります。

  • 甲（こう）／陽・木 … 大樹・まっすぐ伸びる力。
  • 乙（おつ）／陰・木 … 草花・柔軟さ。
  • 丙（へい）／陽・火 … 太陽・外向的。
  • 丁（てい）／陰・火 … 灯火・内なる熱。
  • 戊（ぼ）／陽・土 … 山・大地の安定。
  • 己（き）／陰・土 … 畑・育てる力。
  • 庚（こう）／陽・金 … 刃物・改革力。
  • 辛（しん）／陰・金 … 宝石・洗練。
  • 壬（じん）／陽・水 … 大河・包容力。
  • 癸（き）／陰・水 … 雨露・知恵。

十二支（12種類）

十二支は一年の流れや季節・方位を象徴します。算命学では運勢や性質を細かく読み取る要素です。

  • 子／陽・水／冬の極 … 蓄える力、知恵。
  • 丑／陰・土／冬土用 … 粘り強さ、準備。
  • 寅／陽・木／初春 … 始まりの勢い。
  • 卯／陰・木／仲春 … 調和、柔軟。
  • 辰／陽・土／春土用 … 基盤づくり。
  • 巳／陰・火／初夏 … 情熱、直感。
  • 午／陽・火／盛夏 … 活動・表現。
  • 未／陰・土／夏土用 … 調整・成熟。
  • 申／陽・金／初秋 … 収穫・分析。
  • 酉／陰・金／仲秋 … 整理・結実。
  • 戌／陽・土／秋土用 … 守り・忠実。
  • 亥／陰・水／初冬 … 潜在力・再生。

初心者にとっては「十干＝性質の型」「十二支＝時間の流れ」と理解すると分かりやすいです。二つを組み合わせた六十干支は性格診断や運勢占い、相性判断、結婚運や仕事運の鑑定に活用されます。

命式（生年月日から導かれる図）とは何か

算命学における最も重要なツールが命式（めいしき）です。生年月日から十干と十二支を割り出し、年柱・月柱・日柱・時柱に配置します。

  • 年柱：祖先や家系からの影響、幼少期。
  • 月柱：青年期や社会的立場、親との関係。
  • 日柱：本人の本質・性格（日干＝日主）。
  • 時柱：晩年の運、子孫との関係。

命式の実例

例：1985年3月15日 午前10時生まれ

  • 年柱：乙丑（陰の木・陰の土）
  • 月柱：丁卯（陰の火・陰の木）
  • 日柱：庚申（陽の金・陽の金）
  • 時柱：壬午（陽の水・陽の火）

この命式を分析すると、金が中心にあるため判断力が際立ちつつ、火との関係性で葛藤が生じやすい傾向が見られます。木と火の循環が強いため、成長や発展への意欲が高い人物像が浮かびます。

命式は一見複雑ですが、陰陽五行の関係を一つずつ見ていくことで、その人の宿命と運命のシナリオが理解できるようになります。算命学の学びは、自己理解や人生相談、相性占いに直結する実用的な知恵として注目されています。
